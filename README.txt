This repository contains KNIME workflows created as part of MCA (AI & ML) coursework.
Each assignment uses the adult.csv dataset from the UCI Machine Learning Repository.

Assignment 1 — KNIME Workflow
1. Read adult.csv from KNIME Hub.
2. Count + average age of women with income >50K.
3. Average of all numerical columns for 4 sex–income groups.
4. Missing value calculations for Occupation & Marital-status.

Assignment 2 — Aggregation + Join
1. Read adult.csv.
2. Compute average age & count for sex-income groups.
3. Join aggregated values back to original dataset.

Assignment 3 — Row Filtering + Concatenation
1. Read adult.csv.
2. Extract age 20–40 with workclass starting with 'S'.
3. Extract age 40–60 with workclass starting with 'P'.
4. Concatenate both subsets.

Tools Used:
- KNIME Analytics Platform
- CSV Reader, Row Filter, GroupBy, Joiner, Concatenate nodes
